What is Inline CSS?

Inline css is used to apply specific types of style attributes for a particular html element. This can be used when you want to add or change only a few elements within your webpage.

What is Internal CSS?

Internal css is used when one single html page has it's own specific style. The style would be defined inside the head section of the html page, and it can be used if you would like to change many elements within a webpage at once.

What is External CSS?

External css is used when you want to change your website by using one file. You would have to create a link which contains all of your files. This link would be able to be used within multiple of your webpages.
