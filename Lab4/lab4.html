Lab #4

Tables and Colors
By: Alena Persaud

To make a table, you would need to write tr style="border:solid;border-width:2px" to create it and then you would write td style="border:solid;border-width:2px" and copy and paste it three times. For the first line you would write the color, the second line you write the hexadecimal, and for the third line you write the rgb. Then, for the last line, you would write td style="background-color:" and write the name of that color in lowercase, this will insert the actual color into the table and that will be the hue.
